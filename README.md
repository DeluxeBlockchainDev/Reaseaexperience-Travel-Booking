# RedSeaExperience Travel Booking Website

## 1. B2C

#### For run

Create "redsea_experience" db with "utf8mb4_general_ci" charset in mysql.

And then 

```bash
>> composer install
>> copy .env.example .env
>> php artisan key:generate
>> php artisan migrate
>> php artisan serve
```

Go to "localhost:8000"
